# Laptop Refresh Policy

## Overview
This document provides detailed technical support information for **Laptop Refresh Policy**. It is intended for end-users and support staff.

## Configuration & Instructions
## Laptop Refresh Policy

We want to ensure every employee has performant hardware to do their best work.

### Eligibility

*   **Refresh Cycle**: Every 3 years.
*   **Early Refresh**: Considered for hardware failure or significant role change requiring different specs.

### Standard Configurations

#### Engineering & Design
*   **Mac**: MacBook Pro 16", M3 Max, 64GB RAM, 1TB SSD.
*   **PC**: Dell XPS 15, i9, 64GB RAM, 1TB SSD, NVIDIA GPU.
*   **Linux**: Dell Precision (Developer Edition), Ubuntu pre-loaded.

#### Business & G&A
*   **Mac**: MacBook Air 15", M3, 16GB RAM, 512GB SSD.
*   **PC**: Dell Latitude, i7, 16GB RAM, 512GB SSD.

### Process

1.  IT will notify you 30 days before your eligibility date.
2.  Fill out the [Laptop Refresh Form](https://it.company.com/refresh).
3.  Select your preferred OS and shipping address.
4.  Once the new laptop arrives, you have 14 days to migrate your data.
5.  Return the old laptop using the provided shipping label.

### Data Migration

*   Use **Box** or **Google Drive** to back up your files.
*   Do not use Time Machine for full system restores across different hardware generations.
*   Run the "Migration Assistant" script provided by IT to move local settings.

## Troubleshooting Guide
1.  **Reproduce:** Try to reproduce the issue consistently.
2.  **Isolate:** Determine if the issue is with the hardware, software, or network.
3.  **Logs:** Check the logs for specific error codes.

## Getting Help
If the issue persists:
1.  Search the Knowledge Base.
2.  Ask in the `#hardware-support` Slack channel.
3.  Open a Jira ticket.
