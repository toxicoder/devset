## Hardware Accessories

Employees are entitled to standard peripherals to create a productive workspace.

### Standard Issue Items

You can request these items via the Service Portal without manager approval:
*   **Monitor**: 1x 27" 4K Monitor (Dell/LG).
*   **Keyboard**: Apple Magic Keyboard or Logitech MX Keys.
*   **Mouse**: Apple Magic Mouse or Logitech MX Master 3.
*   **Headset**: Jabra Evolve or equivalent.
*   **Adapters**: USB-C Hub, HDMI cables.

### Special Requests

Items requiring manager approval:
*   Additional monitors (up to 2 total).
*   Vertical mouse / Ergonomic keyboards (Kinesis, ErgoDox).
*   Standing desk converters.
*   Privacy screens.

### How to Request

1.  Go to the [Service Portal](https://support.company.com).
2.  Navigate to **Service Catalog** -> **Hardware**.
3.  Select the item and add to cart.
4.  Checkout. If approval is needed, your manager will be notified via email.

### Returns

If you leave the company or upgrade, accessories over $50 value (monitors, docks) must be returned. Small peripherals (cables, old mice) do not need to be returned.
