# Security Awareness

## Course Overview
**Course Title:** Security Awareness
**Level:** Intermediate/Advanced
**Format:** Self-paced Reading & Hands-on Labs

## Learning Objectives
*   Articulate the core principles of Security Awareness.
*   Apply best practices to real-world engineering scenarios.
*   Troubleshoot common issues and optimize performance.

## Core Curriculum
## Phishing
Phishing is the most common attack vector.
* **Verify the Sender**: Check the email address carefully.
* **Don't Click Links**: Hover over links to see the actual URL.
* **Report Suspicious Emails**: Use the "Report Phishing" button in your email client.

## Password Hygiene
* **Use a Password Manager**: 1Password or LastPass.
* **Enable MFA**: Multi-Factor Authentication is mandatory for all internal systems.
* **Unique Passwords**: Never reuse passwords across sites.

## Physical Security
* **Badge Access**: Never let anyone tailgating into the building.
* **Lock Your Screen**: Win+L (Windows) or Ctrl+Cmd+Q (Mac) when stepping away.
* **Clean Desk Policy**: Do not leave sensitive documents on your desk.

## Hands-on Exercises
1.  **Review:** Read the core curriculum carefully.
2.  **Apply:** Identify one area in your current project where this applies.
3.  **Prototype:** Create a small proof-of-concept in your sandbox environment.
