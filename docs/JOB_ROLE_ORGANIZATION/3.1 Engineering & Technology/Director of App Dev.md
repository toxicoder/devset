# Director of App Dev

**Role Code:** SWEN0003

## Job Description
The Director of Application Development leads the software engineering teams responsible for building the company's customer-facing products. They oversee Backend, Frontend, and Mobile engineering. They focus on feature delivery, architectural integrity, and developer productivity.

## Responsibilities

* **Product Delivery:** Partner with Product Management to deliver roadmap items on schedule.
* **Architecture:** Ensure the application architecture is modular, scalable, and maintainable.
* **Tech Debt:** Balance new feature development with refactoring and tech debt paydown.
* **Developer Experience:** Ensure tools and processes enable engineers to move fast.
* **Team Management:** Manage Engineering Managers and Staff Engineers.

### Role Variations

#### The Feature Factory Director
Focused on high-velocity shipping. Metrics are number of PRs, features released, and cycle time.
**Average Daily Tasks:**
* 09:00 Standup Review
* 11:00 Roadmap Sync with PM
* 14:00 Release Planning
* 16:00 Demo Review

#### The Quality Director
Obsessed with bugs, test coverage, and code reviews. "Zero bug policy."
**Average Daily Tasks:**
* 09:00 Bug Triage
* 11:00 QA Sync
* 14:00 Code Quality Review
* 16:00 Testing Strategy

#### The Mobile/Frontend Specialist
Background in UI/UX engineering. deeply cares about pixel perfection and app performance.
**Average Daily Tasks:**
* 09:00 Design Review
* 11:00 Mobile Release Prep
* 14:00 Frontend Arch Sync
* 16:00 App Store Review

## Common Partners
VP Engineering, Director of Infrastructure, VP Product
